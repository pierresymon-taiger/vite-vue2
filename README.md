## vite-vue2
Sample vue 2 app using vite

## Setup
### .env
Create `.env` for environment varialbles.
```
VITE_APP_KEY=123
etc . .
```

## Scripts
Install and serve
```
// Install
yarn install

// Serve
yarn dev

// Build creates a dist folder
yarn build

```
